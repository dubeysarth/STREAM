# Rebuttal Asset Notes

The rebuttal asset builder renders:

- cross-tier outlet summary tables,
- topology-off delta tables,
- zero-shot India failure-patch summaries,
- curation-sensitivity tables,
- per-reviewer markdown snippets keyed to those outputs.

The repository keeps the expected artifact names stable through the JSON
templates and support markdown so reviewer-facing material can be regenerated
from one shared layout instead of a one-off local directory structure.
